Shipped UI notes (Win32)

Authoritative product spec remains README.md. This page is the visual reference for the paired main window.

Controls

AreaImplementation
Status stripPainted in paint.rs, inset M horizontally with CONTENT_TOP_PAD (14px) above. Connection only (Connected / Offline).
Sync footerAll synced, Checking…, batch done/total · pct%, failures, Retry failed.
Server rowIDC_SERVER_HDR, IDC_SERVER_URL_LABEL, IDC_PAIR_DEVICE grey button (Connect / Reconnect).
Backup folderEditable IDC_WATCH_FOLDER + Open + Browse.
Server destinationRead-only — dest_path_rect painted panel (#fafafa / #e0e0e0) + IDC_REMOTE_FOLDER static text. Not an edit control.
Recent activityOwner-draw IDC_ACTIVITY_LIST: 9pt rows, 8pt status, scrollbar gutter. Failed uploads pin to top (red, error detail). Retry failed in sync footer.
Sync footerIDC_SYNC_STATUS, IDC_SYNC_PROGRESS, IDC_SYNC_ETA.
Meta footerVersion link · GitHub icon button · Update (hidden until available) · Rui Almeida right.

Shared button size

All grey/owner-draw pushbuttons use ACTION_BTN_W × ACTION_BTN_H (76×26 px, height matches folder input).

Auto-save (no Save)

  • persist_settings on Browse + checkbox toggles.
  • Pair success still saves and starts sync via on_app_pair_result.
