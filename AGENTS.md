# Repository Instructions

- After every code change, rebuild the app and relaunch it.
- Always report whether the rebuild succeeded and whether the relaunched app is running.
- Use Visual Studio Build Tools 2022 to build.
- Build command:
  `cmd /c "call \"C:\Program Files (x86)\Microsoft Visual Studio\2022\BuildTools\Common7\Tools\LaunchDevCmd.bat\" -arch=x86 && \"C:\Program Files\CMake\bin\cmake.exe\" --build build --config Release"`
